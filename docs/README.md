# 文档索引

本目录文档按“自动产物 + 写作模板”组织。

## 自动产物

- `experiment_report.md`：由 `python3 scripts/cli.py report` 自动生成。

## 写作模板

- `thesis_experiment_section_template.md`：论文实验章节模板（路线 B 口径）。
- `defense_full_deck_template.md`：答辩整套页模板（单文件）。

## 使用建议

1. 先跑完整实验链路并生成最新 `experiment_report.md`。
2. 再将报告中的关键数字填入模板，避免手动抄写错误。
3. 若配置结构调整，先更新 README 与本索引，再更新模板内容。
